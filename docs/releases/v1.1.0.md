# v1.1.0

Minor release for GitHub Actions dependency maintenance.

## Included

- Updated CI and release workflows to use `actions/checkout@v6`.
- Updated CI and release workflows to use `actions/setup-node@v6`.
- Bumped package and plugin manifest versions to `1.1.0`.

## Package Boundary

Release packages include only runtime skill files and plugin manifests. They exclude raw research, intake material, local fixtures, repository docs, workflows, and temporary build output.

## Validation

Run before publishing:

```bash
npm run validate
npm run package -- v1.1.0
```
