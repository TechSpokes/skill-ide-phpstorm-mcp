# v1.0.0

First public release of the TechSpokes `phpstorm-mcp` skill.

## Included

- Runtime `SKILL.md` for PhpStorm MCP tool routing in PHP projects.
- References for search, inspections, quick fixes, refactoring, execution, Composer runtime, database safety, and IDE actions.
- Codex and Claude plugin manifests with TechSpokes author metadata.
- Repository maintenance documentation for future capability sweeps.

## Package Boundary

Release packages include only runtime skill files and plugin manifests. They exclude raw research, bootstrap files, local fixtures, repository docs, workflows, and temporary build output.

## Validation

Run before publishing:

```bash
npm run validate
npm run package -- v1.0.0
```
