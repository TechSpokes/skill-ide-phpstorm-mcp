# v1.0.2

Patch release for plugin manifest metadata.

## Included

- Updated Codex and Claude plugin descriptions from PHP-only project wording to PhpStorm project wording.
- Added `jetbrains` to plugin manifest keywords.
- Bumped package and plugin manifest versions to `1.0.2`.

## Package Boundary

Release packages include only runtime skill files and plugin manifests. They exclude raw research, intake material, local fixtures, repository docs, workflows, and temporary build output.

## Validation

Run before publishing:

```bash
npm run validate
npm run package -- v1.0.2
```
