# Focus Ledger

Focus Ledger is a small PHP sample app for testing the PhpStorm MCP server.

It intentionally avoids Composer and keeps simple PHP syntax so it remains a stable fixture while PhpStorm project configuration changes. The project was originally observed at PHP language level 5.6, then later changed to PHP 8.2.

## Commands

```powershell
C:\wamp64\bin\php\php8.2.29\php.exe .intake/app/tests/run.php
C:\wamp64\bin\php\php8.2.29\php.exe .intake/app/cli.php list
C:\wamp64\bin\php\php8.2.29\php.exe .intake/app/cli.php add "Write MCP notes" high
```
